cusplinux.org
cusplinux is a highly Alpine-based, personalized Linux distribution. It is developed on GitHub.

What is cusp?

A personalized set of packages and build tooling on top of Alpine's tooling. It is intended for use on VMs for development. Generally it is developed to run on tmpfs using the Alpine backup (that is `lbu`) overlay system.

Why cusp?

Something like Alpine could suggest "peak" or "top" (not sure if that is a part of the Alpine naming choice), where-as I can create a downstream distribution to indicate the disappoint of standing up a any form of 'linux' for my own needs.

Can I use cusp?

I would not, it's solely developed for my personal needs.
