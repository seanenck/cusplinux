cusplinux.org
cusplinux is a highly personalized Linux distribution. It is developed on GitHub.

What is cusp?

It is intended for use on VMs for development. Generally it is developed to run on tmpfs using an overlay system.

Why cusp?

Something like Alpine (which cusp originally spawned from) could suggest "peak" or "top" (not sure if that is a part of the Alpine naming choice), where-as I can create a downstream distribution to indicate the disappointment of standing up a any form of 'linux' for my own needs.

Can I use cusp?

I would not, it's solely developed for my personal needs.
